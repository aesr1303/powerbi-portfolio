# Dashboard Cumplimiento Presupuesto Anual

Descripción del proyecto

Reporte desarrollado en Power BI para el seguimiento del presupuesto de ventas, comparando valores presupuestados versus resultados reales, orientado a áreas comerciales y de planificación.

Objetivo de negocio

Monitorear el cumplimiento del presupuesto de ventas y detectar desviaciones de forma oportuna para apoyar la gestión comercial.

KPI e indicadores analizados

- Ventas reales vs presupuestadas
- Variación absoluta y porcentual
- Cumplimiento del presupuesto (%)
- Evolución temporal del desempeño comercial

Análisis y decisiones que habilita

- Detectar brechas de cumplimiento por período
- Priorizar acciones comerciales correctivas
- Evaluar desempeño del plan de ventas

Herramientas utilizadas

- Power BI Desktop
- Modelado de datos
- Medidas DAX para análisis de variaciones

Posibles mejoras

- Desglose por zona, canal o vendedor
- Automatización del proceso presupuestario
- Integración con incentivos comerciales
