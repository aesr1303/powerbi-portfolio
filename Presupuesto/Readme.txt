# Dashboard Cumplimiento Presupuesto Anual

**Objetivo:**  
Dashboard para realizar análisis del cumplimiento del presupuesto de una compañía distribuida en Tiendas.

**Indicadores principales:**
- Presupuesto
- Ventas
- Variación
- Margen de variación

**Fuente de datos:**  
Dataset de práctica / datos simulados (Datdata).

**Herramientas:**  
Power BI, Power Query, DAX básico.

**Notas:**  
Dashboard creado con fines de aprendizaje y portafolio.
